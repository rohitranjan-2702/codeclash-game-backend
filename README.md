# How to run the backend:

1. Clone the project
   - git clone https://github.com/rohitranjan-2702/random-monorepo-game.git
2. Run `cd apps/backend`
3. Install dependencies
   - `pnpm install`
4. Run the project
   - `pnpm build`
   - `pnpm dev`
5. The websocket server starts listening on ws://localhost:8000
